# Android-Flavors
This is a demo of using Flavors in order to generate two versions of the app with different settings. This app uses MVP in a basic way of a product items.

+ **Technologies used:**
  - Android Flavors
  - Dagger2
  - RxJava
  - Retrofit
  - OkHttp
  - Glide
  - Kotlin
